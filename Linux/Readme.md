## Now... What is linux?

Linux is a powerful, open-source operating system used extensively in servers, cloud environments, and DevOps workflows. It provides a stable, flexible, and secure platform to run applications and manage infrastructure. Learning Linux is essential for any DevOps professional because most cloud servers, containers, and CI/CD pipelines rely on it.

## Some of the things included in linux:

* Command-Line Basics – Navigating the file system, managing files, and using essential commands.
* User & Permission Management – Creating users, managing groups, and setting permissions.
* Process & Resource Management – Monitoring system processes, CPU, memory, and disk usage.
* Networking Basics – Configuring network interfaces, checking connectivity, and troubleshooting issues.
* Shell & Scripting Essentials – Using the shell to automate tasks and interact with the system efficiently.
* Package Management – Installing, updating, and removing software using package managers like apt, yum, or dnf.

## Basic Linux Commands
```
ls        # List files in a directory
cd        # Change directory
pwd       # Show current directory path
mkdir     # Create a new directory
rm        # Remove files or directories
```
